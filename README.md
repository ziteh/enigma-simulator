# Enigma Simulator

An online Enigma machine simulator. Encrypt and decrypt messages using configurable rotors, reflectors, and plugboard settings.

Tech stack:

- React
- Vite
- [shadcn/ui](https://ui.shadcn.com/)
- [Lucide icons](https://lucide.dev/)

## Usage

```bash
pnpm i
pnpm dev
```
